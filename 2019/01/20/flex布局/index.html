flex布局

本次介绍h5开发的重要布局方式————弹性盒子

什么是flexbox

CSS3 弹性盒子(Flexible Box 或 Flexbox)，是一种用于在页面上布置元素的布局模式，使得当页面布局必须适应不同的屏幕尺寸和不同的显示设备时，元素可预测地运行。对于许多应用程序，弹性盒子模型提供了对块模型的改进，因为它不使用浮动，flex容器的边缘也不会与其内容的边缘折叠。

使用flex布局

1.弹性容器

包含着弹性项目的父元素。通过设置 display 属性的值为 flex 或 inline-flex 来定义弹性容器。

2.弹性项目

弹性容器的每个子元素都称为弹性项目。弹性容器直接包含的文本将被包覆成匿名弹性单元。

3.一些属性

1.flex-direction 确立主轴 使得容器内按照行还是列方向排列 参数为：row/column
2.justify-content 定义了在当前行上，弹性项目沿主轴如何排布
3.align-items 定义了在当前行上，弹性项目沿侧轴默认如何分布
4.align-self 定义了单个弹性项目在侧轴上应当如何对齐，这个定义会覆盖由 align-items 所确立的默认值。
5.flex-wrap 弹性项目可以排布在单个行或者多个行中。此属性控制侧轴的方向和新行排列的方向
6.flex-grow 扩展子元素的宽度 设置当前元素应该占据剩余空间的比例值
比例值计算：当前空间的flex-grow/所有兄弟元素的flex-grow的和
7.flex-shrink 定义收缩比例，通过设置的值来计算收缩空间 默认值为1
比例值计算：当前空间的flex-shrink/所有兄弟元素的flex-grow的和
8.flex属性 包括flex-grow 和flex-shrink 默认为 0 1 auto
